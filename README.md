# Project 1: DecodeBot AI (Simple Chatbot)

An elegant, desktop-based conversational application featuring a modern graphical user interface built with Tkinter. The backend utilizes structural pattern matching and natural language parsing via regular expressions.

---

## 🛠️ Key Features & Stack
- **GUI Framework:** Python Tkinter (Modern fluid layout with `#1e3a8a` branding theme).
- **Architecture:** Tokenized rule-based keyword routing with session safety checks.
- **Dynamic Functions:** Integrated context mapping for daily utility questions like date, time, academic management, and anxiety management.
- **UX Features:** Automated auto-scroll mechanics on message delivery and chat boundary tracking.

---

## 📂 Structural Artifacts
- `simple chatbot.py` — Core entry script containing user processing modules and Tkinter frontend render parameters.
- `chat_history.txt` — Log structure documenting temporal chat interactions.

---
